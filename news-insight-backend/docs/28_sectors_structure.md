## 참고사항

1. **총 28개 섹터**로 구성되어 있으며, 각 섹터는 투자 분석 목적으로 분류됨
2. **KRX 업종 매핑**: 현재 132개 KRX 업종이 28개 섹터로 매핑됨
3. **Confidence 레벨**:
   - HIGH: 명확한 매핑 규칙 (31개, 23.5%)
   - MEDIUM: Industry 컬럼 기반 보정 (100개, 75.8%)
   - LOW: 추가 확인 필요 (1개, 0.8%)
4. **매핑이 없는 섹터**: SEC_GAME, SEC_TRAVEL, SEC_CONSUMER, SEC_CARD는 KRX 업종 분류에서 직접적으로 매핑되지 않음 (DART 데이터 기반 분류 필요)
