# 배치 수집 모니터링 가이드

## 현재 진행 상황

배치 수집이 백그라운드에서 실행 중입니다.

## 모니터링 방법

### 1. 실시간 로그 확인
```powershell
# PowerShell에서 실행
Get-Content data\batch_100_log.txt -Tail 30 -Wait -Encoding UTF8
```

### 2. 진행 상황 확인
```bash
python scripts/check_progress.py
```

### 3. 로그 파일 직접 확인
```
파일: data/batch_100_log.txt
```

## 체크 포인트

### 자동 체크
- ✅ **20개마다** 중간 체크 (빠진 기업 확인)
- ✅ **100개 완료 시** 최종 리포트 생성

### 수동 체크
- 로그 파일에서 진행 상황 확인
- `check_progress.py` 실행하여 전체 통계 확인
